# ansible-mddoc

ansible-mddoc is a package used to auto generate documentation for an ansible role and playbook. 

## To install
```
pip install ansible-mddoc
```


## To run

Call ansible-mddoc passing in the path to the role
```
ansible-mddoc <path_to_role>
```
